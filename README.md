# EV DATA ANALYSIS (INTERACTIVE DASHBOARD USING TABLEAU)
## Project objective
To analyze electric vehicle (EV) data to understand adoption trends, vehicle types, geographic distribution, and key manufacturers, using Excel and Tableau to create an interactive dashboard that supports data-driven insights into the EV market.

# DATA USED
<a href ="https://github.com/ishuravi001/EV-DATA-ANALYSIS/blob/main/Electric_Vehicle_Population_Data.xlsx"> Dataset </a>

## KPI Questions:
-	What is the total number of electric vehicles currently registered?
-	What is the average electric range across all EVs?
-	How do BEV and PHEV vehicles compare in overall count and share?
-	How has EV adoption grown over the years by model year?
-	Which states have the highest number of electric vehicles?
-	Which manufacturers lead the EV market by total vehicles?
-	What percentage of vehicles are CAFV eligible vs not eligible?
-	Which EV models contribute the most to total vehicle volume?
  
## Process 
-	Collected and reviewed raw electric vehicle data from the source.
-	Cleaned and organized the data in Excel by handling missing values and standardizing fields.
-	Performed basic calculations and created KPI metrics in Excel.
-	Imported the cleaned data into Tableau for visualization.
-	Designed an interactive dashboard with filters, charts, and maps to analyze trends and insights.
